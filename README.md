# 🧹 File Organizer v5.1

Умная утилита для автоматической сортировки файлов с графическим интерфейсом!

## ✨ Возможности:
- 📄 Сортировка по типам файлов (настраивается)
- ⚙️ Конфигурационный файл `config.json`
- 📝 Подробное логирование всех действий
- 🛡️ Защита от ошибок (не падает при проблемах)
- 🎨 Красивый графический интерфейс (GUI)
- 📦 Отдельный .exe файл (не нужен Python!)
- 🎨 Уникальная иконка приложения

## 🚀 Как использовать:

### Для пользователей Windows:
1. Скачайте `gui.exe` и `config.json` из раздела Releases
2. Положите их в одну папку
3. Запустите `gui.exe`
4. Выберите папку для уборки и нажмите "Запустить"

### Для разработчиков:
```bash
pip install customtkinter
python gui.py
```
