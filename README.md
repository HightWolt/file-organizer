# 🧹 File Organizer

Простая утилита для автоматической сортировки файлов по папкам.

## ✨ Что умеет:
- 📄 Перемещает .txt файлы в папку Texts
- 🖼️ Перемещает .jpg и .png в папку Images
- 📑 Перемещает .pdf и .docx в папку Docs

## 🚀 Как использовать:
1. Положите main.py в папку с файлами
2. Запустите: `python main.py`
3. Наслаждайтесь порядком!
