# 🧹 File Organizer v4.0

Умная утилита для автоматической сортировки файлов с конфигурацией и логированием.

## ✨ Возможности:
- 📄 Сортировка по типам файлов (настраивается)
- ⚙️ Конфигурационный файл `config.json`
- 📝 Подробное логирование всех действий
- 🛡️ Защита от ошибок (не падает при проблемах)

## 🚀 Как использовать:
1. Скачайте `main.py` и `config.json` в одну папку
2. Настройте `config.json` под свои нужды
3. Запустите: `python main.py`
4. Проверьте лог-файл с историей операций
